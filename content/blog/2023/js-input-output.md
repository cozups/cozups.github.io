## 2. readline 모듈 사용하기

```jsx
const readline = require('readline');
const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout,
});

rl.on('line', (line) => {
  // 줄 바꿈이 입력 될 때 마다 발

	// line을 가공하여 변수에 저장
	...

	// close로 넘어감
	rl.close();
}).on('close', () => {
  // 저장된 변수를 이용하여 계산 후 출력
});
```

⇒ fs 모듈을 사용하는 것이 편하다.
